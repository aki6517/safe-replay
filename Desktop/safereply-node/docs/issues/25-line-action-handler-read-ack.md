### 背景 / 目的

Type B用のアクションボタン（既読・確認メール）の処理を実装し、共有メッセージへの対応を実現する。

- 依存: #20
- ラベル: `backend`, `line`

### スコープ / 作業項目

- `action=read`で既読処理（ステータス更新）
- `action=ack`で確認メール送信処理（Gmail）
- DBへのアクションログ記録
- 完了通知の送信
