### 背景 / 目的

Railwayへのデプロイ設定とGitHub連携により、自動デプロイの基盤を構築する。

- 依存: #4
- ラベル: `infra`, `deployment`

### スコープ / 作業項目

- Railwayプロジェクトの作成
- `railway.toml`の設定
- GitHubリポジトリとの連携
- 環境変数の設定
- 自動デプロイの動作確認

### ゴール / 完了条件（Acceptance Criteria）

- [ ] `railway.toml`を作成

- E2E: Railwayにデプロイ後、本番環境のヘルスチェックエンドポイントにアクセス
- 検証方法: `git push`で自動デプロイが実行され、本番環境が正常に動作することを確認
