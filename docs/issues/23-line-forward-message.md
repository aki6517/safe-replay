### 背景 / 目的

LINE経由で転送されたメッセージを処理する機能を実装し、手動転送による外部連絡の処理を実現する。

- 依存: #8, #16, #17, #18
- ラベル: `backend`, `line`

- 転送メッセージを「外部連絡」として処理
- AIトリアージとドラフト生成の実行
- Type A/Bの場合のみ通知
