### 背景 / 目的

LINE Botのアクションボタン（送信・修正・断る）の処理を実装し、ユーザーアクションに応じた処理を実現する。

- 依存: #19, #10, #11
- ラベル: `backend`, `line`

### スコープ / 作業項目

- Postbackイベントのパース処理
- `action=send`で返信送信処理（Gmail/Chatwork）
- `action=edit`で編集モード開始処理
- `action=dismiss`で却下処理
- DBへのアクションログ記録

### ゴール / 完了条件（Acceptance Criteria）

- [x] Postbackイベントのパース処理を実装
- [x] `action=send`で返信送信処理を実装（Gmail/Chatwork）
- [x] `action=edit`で編集モード開始処理を実装（将来実装予定のメッセージを表示）
- [x] `action=dismiss`で却下処理を実装
- [x] DBにアクションログを記録（メッセージステータスを更新）
- [x] 完了通知をLINE Botに送信
- [x] 動作確認完了（却下・編集アクションが正常に動作）

### テスト観点

- E2E: LINE Botでアクションボタンをタップし、処理が実行されることを確認
- 検証方法: [送信][修正][断る]ボタンをタップし、それぞれの処理が正常に動作することを確認
